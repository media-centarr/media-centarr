Page redistribution — visual mockups

Self-contained HTML mockups exploring a redistribution of pages and features across the app. The driving observation: the current "Library" page does three different jobs (Continue Watching, Browse, Upcoming) that pull in three different mental directions. The proposal splits those into focused single-purpose pages and reorganises the sidebar into Watch and System groups — a content-first cinematic frontstage and an operator backstage.

Click any mockup below to see the full page. Each mockup includes the proposed sidebar, so you can click between them. Each folder also has a REASONING.md documenting the design decisions.
