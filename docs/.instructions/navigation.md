# Navigation Conventions

## Cross-Doc Navigation Bar

Every doc in `docs/` has a one-line blockquote navigation bar immediately after the purpose statement. Current page is bold (not linked), all others are linked, separated by ` · `.

Doc order: Getting Started · Configuration · Architecture · Watcher · Pipeline · TMDB · Playback · Channels · Library

## Page Table of Contents

Below the nav bar, a bare bullet list of `##`-level headings as anchor links. No heading above the list.

## What NOT to Do

- No `## Contents` heading — unnecessary visual weight
- No back/forward links at the bottom — the nav bar makes them redundant
- No TOC or nav bar in README.md — it's the entry point, not part of the doc sequence
- No duplicate doc index in README — the docs handle their own cross-navigation
