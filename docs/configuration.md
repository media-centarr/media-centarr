# Configuration

End-user configuration has moved to the wiki:

- **[Configuration File](https://github.com/media-centarr/media-centarr/wiki/Configuration-File)** — the TOML file (`watch_dirs`, `exclude_dirs`, `database_path`, `file_absence_ttl_days`).
- **[Settings Reference](https://github.com/media-centarr/media-centarr/wiki/Settings-Reference)** — everything else (TMDB, Prowlarr, playback, pipeline) that lives on the in-app Settings page.
- **[Adding Your Library](https://github.com/media-centarr/media-centarr/wiki/Adding-Your-Library)** — the walk-through for the most-edited setting.

Contributor-internal `MediaCentarr.Config` details live inline in the module's `@moduledoc` (`lib/media_centarr/config.ex`).
